# Tridev A&D Shine

Tridev A&D Shine is a full-stack jewellery e-commerce website built with React, Vite, Node.js, and Express. The website allows customers to browse jewellery products, add products to cart, place orders through WhatsApp confirmation, and track order status using an Order ID and phone number.

The project also includes an admin dashboard where the store owner can manage products, upload product images, update stock, view customer orders, update order status, and contact customers directly through WhatsApp.
